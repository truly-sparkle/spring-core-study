# 스프링 컨테이너와 스프링 빈

스프링은 Application Context라는 컨테이너 (빈들이 담겨있는)가 존재한다.

사용자가 등록한 스프링 빈들을 담아 놓는 컨테이너로 스프링 빈 간의 의존 관계를 다양한 방법으로 (어노테이션 기반 등) 연결해준다.
